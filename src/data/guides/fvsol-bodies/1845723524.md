## 상담 시 활용 프로세스 (5단계)

1. **문의 유형 파악**: 키워드(플레이어·오류·연동·지점·보강·QR·학부모·재종·교재배부) → 대분류 특정
2. **본인 확인**: 회원 정보 미제공 시 → 【케이스 2-4】 정보 요청 템플릿 발송 / 학부모 경우 학생 정보 추가 확인
3. **소관 판단**: 기술 지원(플레이어·앱·계정·영상 상태) ↔ 비기술(수업·강좌·환불·교재) 분기
4. **처리 및 에스컬레이션**: 직접 처리 가능 건은 즉시 대응 / 불가 건은 지점·연구소 안내 + 에스컬레이션 티켓(Jira SERP) 생성
5. **사후 확인**: "로그아웃 후 재로그인" 등 확인 요청 + 추가 문의 안내
