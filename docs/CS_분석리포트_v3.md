## 4. 교차 분석 — 양 채널의 일치·괴리

| 영역 | 고객 (myclass) | 운영자 (단과톡방) | 해석 |
|---|---:|---:|---|
| **계정이관/통합** | 566건 (40%) | **942건 (17%)** | 양쪽 1순위 — 위키 + 프로덕트 동시 개선 P0 |
| **영상/콘텐츠** | **440건 (31%)** | 135건 (2.4%) | **시스템 결함 (운영자 해결 불가)** → 프로덕트 |
| **결제/환불** | 92건 (7%) | **677건 (12%)** | 운영자가 더 많이 핸들링 — 백오피스 처리 비중 큼 |
| **출결/QR** | 60건 (4%) | 402건 (7.1%, 부정 54.5%) | 운영자가 더 고통 — 출석부·QR 시스템 미숙 |
| **수강료/청구** | (분류 안 됨) | 198건 (부정 51.5%) | 정산 영역 — 정산팀 89건 + 지점 협의 |
| **로그인/계정** | 32건 (부정 46.9%) | (권한·접근 카테고리) | OKTA·인증 흐름 UX 마찰 |

### 핵심 인사이트

1. **양 채널 1순위 모두 "계정이관/통합"** — 학원 회원 시스템 마이그레이션이 진행 중이라는 명확한 신호
2. **영상 재생은 직원이 손도 못 댐** (운영자 채널 2.4% vs 고객 31%) — **순수 프로덕트 이슈**
3. **출결/수강료가 운영자 부정감정 50%+** — 위키만으로 안 풀림, 시스템 안정화 동반 필요
4. **운영자 톡방 12-1월 peak (66%)** + 고객 채널 12-1월 peak (58%) — **양 채널 시즌 동조**
