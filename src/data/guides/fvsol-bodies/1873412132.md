## 0. 결제 체계 구조 (PG / VAN)
AMS 결제 체계는 PG(온라인)와 VAN(오프라인 단말기/키오스크) 이원 구조로 운영됩니다.

### PG (온라인)
- MID 구성: 단과(대치/목동/반포/분당), 재종(대치/목동), 기숙(용인), 라이브, 출판, 법인, 원격(라이브) — 총 9개 MID
- 확인 경로: 스마트로 Admin에서 MID 기준 조회 (여러 MID 합산 조회 가능)

### VAN (오프라인 단말기/키오스크)
- TID (사업자 단위 단말기 번호)
- 단과/재종 구분: 동일 TID 내 가맹점번호로 구분 (단과: 가맹점 A, 재종: 가맹점 B)
- 키오스크(PS Plus): UI로 7개 지점 구분, 결제는 9개 TID 사용
- VAN 확인 경로: 스마트로 가맹사이트(스마일비즈) > 거래관리 > 신용카드

### 수강료 / 교재 사업자 구분
- 수강료: 학원사업자 (MID/TID)
- 교재: 출판사업자 (MID/TID)
- ⚠️ 사업자가 다르면 합산 결제 불가 — 수강료 + 교재 동시 결제 시 각각 다른 MID/TID로 별도 처리

### 현재 운영 사업자 (8개)
대치, 목동, 분당, 반포, 용인, 출판, 법인, 원격학원(라이브)

### 관계사 구조
- 스마트로: PG 운영사 + VAN 사업 주체
- 오케이정보통신: 단말기 제조/세팅, 현장 설치, 스마트로와 소통
- 키오스크(PS Plus): 주문 제작, 월 이용료 1만원, 소통은 오케이정보통신 경유

## 1. 전환결제 (가상계좌 / 카드 / 단말기 / 현금)
온라인 전환 시 결제요청 URL 발송. 새 결제 완료 → 기존 결제 자동으로 환불대기 상태 변경.

### 기존 결제 처리 결과 (전환 후)
- PG 카드: 자동 환불완료 (추가작업 불필요)
- VAN 단말기/VAN키오/현금/가상계좌: 환불대기 상태 → 환불상세에서 반드시 환불 처리(이체완료) 필요
- 가상계좌 전환: 신규 결제 = '입금대기' → 입금완료 시 기존 결제가 환불대기로 전환

### 전환결제 불가 케이스
- 입금대기 건: "입금대기 상태의 결제건은 전환결제가 불가합니다"
- 500원 미만: "500원 미만 결제건은 전환결제가 불가합니다"
- 전환 진행 중(입금대기): "가상계좌로 전환결제 진행 중입니다"
- 결제취소 건: "환불 가능한 금액이 없는 결제건은 전환결제가 불가합니다"

### ⚠️ 전환결제 유의사항
- 전환결제 건은 환불취소가 불가능 (가장 중요)
- 금액 변경 불가 (동일 금액만 가능, 변경 시 환불+재결제 필요)
- 할부 건 전환 시 잔여 할부 금액 기준 처리
- 전환 후 스마트로 Admin에서 취소/재결제 내역 확인 필수
- 가상계좌 전환 의사 철회 시 결제상세에서 '결제취소' 처리
- 전환결제 URL 유효기간: 7일

## 2. 500원 미만 결제 / 0원 결제 방법

### 판매가 0원
입반 시 판매가 0원 회차는 청구 생성 여부와 무관하게 출석처리 가능. 청구 생성 시 즉시 납부완료 처리.

### 100% 장학혜택 쿠폰 적용
- AMS: '0원결제' 선택 후 [결제요청] 클릭 → 납부완료
- MYCLASS/학부모앱: [납부하기] 클릭 → PG 이동 없이 납부완료
- ⚠️ 0원결제 환불 시 주의: 결제금액이 0원이므로 최초 청구 회차만큼 환불. 이용하지 않은 금액만큼만 환불 후 추가 청구 + 0원결제 필요.

### 소액 결제 (1원~499원)
- 1원~99원: 현금 가능 (MYCLASS/KIOSK/학부모앱에서 "관리자에게 문의해 주세요" 얼럿)
- 100원~499원: 카드 가능
