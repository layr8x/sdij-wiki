## 기능 소개
오프라인 플랫폼의 청구/수납 관리 시스템은 강좌 수강료 및 교재비 등의 청구 생성부터 납부 완료까지의 전체 프로세스를 통합 관리합니다.

- **청구 관리**: 수강료, 연결교재 청구 생성 및 관리
- **납부 관리**: 납부 현황 조회 및 미납 관리
- **결제 관리**: 결제 내역 조회 및 처리
- **환불 관리**: 환불 요청 및 처리
- **신한캠퍼스 처리**: 무통장입금 매칭 및 환불
- **배부 관리**: 물류상품 배부 현황 조회 (26년 2월 오픈)
- **배송 관리**: 택배 배송 현황 관리 (26년 2월 오픈)

## 4. 청구/수납 관리

### 4.1 청구 관리 > 목록 조회
생성한 청구 목록을 확인. 신규 청구를 생성하거나 청구상태/납부기한을 변경 가능.

- **청구상태**: 청구완료 상태에서만 FRONT 화면 노출. 청구취소로 변경 = 복구 불가능(삭제)
- **납부기한**: 운영 일정이 변경된 경우 납부기한 수정 또는 청구취소 후 재생성
- **미납관리**: 납부코드가 없는 청구만 필터링 → 미납인 청구 목록 확인

### 4.2 청구 관리 > 결제 요청 URL 발송
선택한 청구 건을 학생/학부모에게 간편납부 URL을 SMS로 발송.

**SMS 발송 내용:**
```
[시대인재 지점명] 등록 안내
이름 학생/학부모님 귀하
상품명 등록 기한 YYYY-MM-DD
즉시 납부를 원하시면 아래 링크를 통해 결제 가능합니다.
▶ 납부하기: URL
◦ 유효기간: 발송일 +5일 23:59:59까지
```

### 4.3 청구 관리 > 청구서 생성

**수강료 청구 생성 — 기본 설정:**
- 강좌(반): 선택 버튼 클릭 시 상품조회/선택 화면 호출 (1개 선택 가능). 선택 완료 후 강좌코드 자동 채워짐.
