## 1. 출결/배부 교재 유형별 정책

### 구분
1. 회차컨텐츠 (서바이벌 컨텐츠)
2. 교재단품 (필수 주교재/부교재)
3. 선택컨텐츠 (서바이벌 선택 커리큘럼)

### 기본 개념

**회차컨텐츠 (회차패키지):**
- 청구가 생성되면 배부박스가 생성됨
- 청구 생성 시 해당 교재 배부회차에 출결 박스가 생성된 경우에 한해 교재 청구 생성 가능
- 회차패키지 교재는 강의 회차와 1:1로 묶여 움직이는 교재
- 결석(취소/반납) 처리 시 해당 교재가 배부회차로 연결되어 있고 출결박스가 있는 회차로 배부박스가 이동

**교재단품 (필수 주교재/부교재):**
- 청구가 생성되면 배부박스가 생성됨
- 청구 생성 시 해당 교재 배부회차에 출결 박스가 생성된 경우에 한해 교재 청구 생성 가능
- 교재단품은 여러 회차에 연결될 수 있지만, 배부 상태는 하나로 관리
- 회차별로 별도 배부 상태가 존재하지 않아, 특정 회차에서 미수령한 경우 다음 배부회차가 있는 경우 수령예정으로 유지

**선택컨텐츠 (선택패키지):**
- 청구가 생성되면 배부박스가 생성되며, 청구 생성은 출결박스 유무에 상관없이 입반생이면 가능
- 선택패키지는 수강 필수 교재가 아닌, 선택 구매 교재
- 출석 유무에 상관없이 납부완료된 경우 배부 처리 가능
- 선택패키지는 수강생이 그룹 단위로만 청구 및 납부되며, 배부 처리만 그룹 내 단품 단위로 관리
