## 응답 분기 의사결정 (UX 다이어그램)

```
User 질문
   ↓
[matchOfficialQa] score ≥ 6 ?
   ├─ YES → detection.source = 'sheet'
   │        └─ decideResponseMode(qa.selfSolve)
   │            ├─ 'yes' → BOT_OFFICIAL_QA + BOT_MENU_PATH + BOT_CONFIDENCE + FEEDBACK
   │            ├─ 'no'  → BOT_OFFICIAL_QA + BOT_ESCALATION(slackTitle prefill)
   │            └─ 'partial' → BOT_OFFICIAL_QA + BOT_TEXT(양분 안내) + BOT_ESCALATION
   └─ NO → 기존 INTENT_RULES fallback (9개 의도)
            ↓
       정답 없음 → BOT_ESCALATION(low-confidence)
```

```bash
cd ~/Library/Mobile\ Documents/com~apple~CloudDocs/🕹️\ layr8x/ams-wiki && \
  rm -f .git/index.lock && \
  git commit -F docs/v5-인테그레이션-노트_20260519.md && \
  git push origin main
```

또는 commit 메시지를 짧게 하고 싶다면:

